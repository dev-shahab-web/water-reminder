# Water Reminder Brand Assets

`water-reminder-mark.svg` is the source brand mark for Sprint 000.

The Expo icon and splash configuration still point at raster PNG paths because Expo requires generated image assets for native builds. Export branded PNGs from this SVG into the existing configured paths before store submission:

- `assets/images/icon.png`
- `assets/images/android-icon-foreground.png`
- `assets/images/android-icon-background.png`
- `assets/images/android-icon-monochrome.png`
- `assets/images/splash-icon.png`
