# 0002 Expo

## Status

Accepted

## Context

The starter needs a productive, production-capable React Native foundation with strong ecosystem support.

## Decision

Use Expo SDK 57 as the framework foundation.

## Consequences

- Expo packages are preferred when they satisfy the requirement.
- Native capabilities should use Expo modules before third-party alternatives.
- The project tracks Expo SDK compatibility, including the required Node.js version.
