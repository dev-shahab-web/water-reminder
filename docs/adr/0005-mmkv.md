# 0005 MMKV

## Status

Accepted

## Context

The starter needs fast local key-value storage for small app-level values.

## Decision

Use MMKV for non-secret key-value persistence when the storage milestone is implemented.

## Consequences

- MMKV is accessed only through `src/platform/storage`.
- Screens and components never access MMKV directly.
- Secrets and tokens require a secure storage strategy, not plain MMKV.
