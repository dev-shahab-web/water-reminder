# Design

The design system will be introduced before application modules.

Planned reusable components:

- `AppButton`
- `AppCard`
- `AppInput`
- `AppDialog`
- `AppAvatar`
- `AppToolbar`
- `AppScreen`
- `AppList`
- `AppBadge`
- `AppChip`

Planned design token files:

- `colors.ts`
- `spacing.ts`
- `radius.ts`
- `typography.ts`
- `elevation.ts`
- `motion.ts`
- `opacity.ts`
- `icons.ts`

No screen should rely on hardcoded colors, spacing, typography, radii, elevation, or motion values once the design system exists.
