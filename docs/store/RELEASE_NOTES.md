# Release Notes

## 1.0.0

First public release of Water Reminder.

- Offline-first hydration logging.
- Gentle reminders and pause controls.
- History and statistics.
- Android home-screen widget with quick actions.
- Optional Health Connect hydration sync.
- Local export, import, reset, and delete controls.
- Optional anonymous diagnostics controlled from Settings.

Water Reminder is a general wellness app and does not provide medical advice.
