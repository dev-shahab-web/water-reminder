## Capture Notes

- Use realistic but synthetic hydration data.
- Do not show Play Console, Firebase, logs, debug menus, or raw exceptions.
- Do not use claims like medically accurate, prevents dehydration, or guaranteed reminders.
- If using a device frame, use one consistent Android frame style.
