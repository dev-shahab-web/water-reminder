# Empty States

## Purpose

Empty states should explain what is missing, reassure the user, and provide the next useful action. They should never shame the user.

## Global Pattern

Every empty state defines:

- Illustration.
- Primary message.
- Secondary message.
- Primary CTA.

## Home: No Water Logged Today

Illustration:

- Simple water glass or droplet.

Primary message:

- "No water logged yet today."

Secondary message:

- "Add your first drink to start today's progress."

Primary CTA:

- First quick-add amount.

## Today's Log: No Entries

Illustration:

- Simple list or glass outline.

Primary message:

- "No entries yet."

Secondary message:

- "Water you log today will appear here."

Primary CTA:

- "Add water."

## History: No History

Illustration:

- Simple calendar.

Primary message:

- "No history yet."

Secondary message:

- "Your logged days will appear here after you start tracking."

Primary CTA:

- "Go to Home."

## History: Filter Has No Results

Illustration:

- Calendar with empty range.

Primary message:

- "No entries in this range."

Secondary message:

- "Try another date range or return to recent days."

Primary CTA:

- "Show recent days."

## Statistics: Not Enough Data

Illustration:

- Simple chart outline.

Primary message:

- "Not enough data yet."

Secondary message:

- "Log water for a few days to see trends."

Primary CTA:

- "Go to Home."

## Reminder Settings: Notifications Not Enabled

Illustration:

- Quiet reminder bell.

Primary message:

- "Reminders are off."

Secondary message:

- "You can track water manually or turn on gentle reminders."

Primary CTA:

- "Turn on reminders."

## Quick-Add Settings: No Active Options

Illustration:

- Simple plus button.

Primary message:

- "No quick-add sizes enabled."

Secondary message:

- "Add at least one size to keep logging fast."

Primary CTA:

- "Add quick size."

## Permission Recovery: Permission Denied

Illustration:

- Reminder bell with slash.

Primary message:

- "Notifications are blocked."

Secondary message:

- "You can still track water manually. To receive reminders, enable notifications in system settings."

Primary CTA:

- "Open Settings", if supported.

Fallback CTA:

- "Keep tracking manually."
